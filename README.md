# onelol1n  
- Payload for patching kernel on iOS 10 iBoot  
by [sakuRdev](https://twitter.com/sakurdev)[(by dora2ios)](https://github.com/dora2-iOS)  
  
This is a payload for apply patch to kernel on the iBoot, and booting it.
